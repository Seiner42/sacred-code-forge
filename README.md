# sacred-code-forge

Частная кузница кода.

Здесь хранятся именно кодовые изделия, которые создаёт Техножрец по поручению Макса:
- утилиты
- скрипты
- прототипы
- эксперименты
- маленькие сервисы
- автоматизация

## Структура

- `projects/` — более цельные проекты
- `scripts/` — одноразовые и служебные скрипты
- `experiments/` — пробы, концепты, черновые механизмы
- `tools/` — небольшие инструменты общего назначения
- `docs/` — заметки, спецификации, планы и описания

## Принцип

Душа агента, память и его внутренние своды здесь не хранятся.
Этот репозиторий предназначен только для кодовых изделий.

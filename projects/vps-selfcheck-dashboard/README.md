# VPS Selfcheck Dashboard

Простой MVP-макет страницы самодиагностики VPS и агентного контура.

## Суть

Этот проект — заготовка под внутренний operational dashboard, который должен показывать:
- состояние VPS;
- состояние Docker-контейнеров;
- состояние OpenClaw / gateway;
- ключевые сигналы тревоги по сервисам.

## Текущий статус

Сейчас это статический HTML-макет экрана, собранный для быстрого просмотра формы MVP.

## Что внутри

- `index.html` — визуальный mock одной страницы.

## Следующий шаг

Превратить статический макет в живую страницу, которая тянет host-level метрики, а не контейнерные.
